# LocalClaw

**AI Agent Deployment for Local Businesses — Secured with NVIDIA NemoClaw**

LocalClaw deploys and manages OpenClaw AI agents for local businesses. Combines OpenClaw's autonomous agent framework with NVIDIA NemoClaw enterprise security.

## Stack
- React (JSX)
- Lucide React icons
- Cormorant Garamond + Inter typography
- No build tooling required — runs on Base44
